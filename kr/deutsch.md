## 독일어 수업 개요

| 항목 | 내용 |
|------|------|
| 대상 | 7세 이상 |
| 단계 | A1-B2 |
| 형식 | 1:1 또는 1:2 온라인 (Zoom)|
| 수업료(1:1) | 60분당 30,000 원 |
| 수업료(1:2) | 60분당 45,000 원 |
| 수업 횟수 및 시간 | 1회 90분 또는 120분, 주 2-3회 권장<br>※회화 수업만 진행 시 1회 60분 수업 가능) |
| 자료 | PDF 자료, 필요 시 녹화 영상 제공 |

### 대상
- 독일어를 처음 배우는 학습자
- 독일 음대 및 미대 지원 예정자
- 교환학생, 구직, 오디션 면접을 준비하는 학습자
- 한국독일학교, 독일 현지 초등학교 편입학을 준비하는 학습자

### 준비가능한 시험 종류
- Telc A2-B2
- Goethe Zertifikat Deutsch A2-B2

### 수업 방식
- 1:1 또는 1:2 온라인 수업
- 정식 수업 시작 전 40분 무료시범수업 진행
- 맞춤형 자료, 과제 및 피드백 제공
- 시험 준비 시 다른 학생 분과의 말하기 연습 주선

- 공용 PDF 자료

## 음악이론 수업 개요

| 항목 | 내용 |
|------|------|
| 대상 | 중·고등학생 / 대학생 / 성인 |
| 범위 | 기초화성학·분석·집합이론 등|
| 형식 | 1:1 또는 1:2 온라인 (Zoom) |
| 수업 시간 | 1회 60분 / 90분 (선택) |
| 수업료(기초, 1:1) | 60분당 40,000원|

### 대상
- 전공수업(화성학, 음악분석 등) 대비가 필요한 중·고·대학생
- 대학원 자격시험 대비가 필요한 학습자
- 논문용 분석과 관련하여 지도가 필요한 학습자

### 수업 방식
- 1:1 또는 1:2 온라인 수업
- 수준 진단 후 커리큘럼 조정
- 과제 및 피드백 제공

### 수업 자료
- 공용 PDF 자료
- 필요 시 맞춤형 과제 제공
